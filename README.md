# Real-Time-LLM-Coherence-Harness-SDE-Bands-Zero-Drift-Lock-
Real-time coherence engine for LLM conversations. Scores every response using TF-IDF + Jensen-Shannon Divergence, tracks the trajectory with a Kalman filter, models variance with GARCH(1,1), and injects corrective directives before the next turn. Detects behavioral and hallucination proxy signals live.
